# General

### 安裝 Flutter

1. 開啟 VSCode
2. 安裝擴充功能 `Flutter`
3. 在 VSCode 中按 `Ctrl` + `Shift` + `P` 輸入 `Flutter` 選擇 `New Project`
4. 選擇下載 SDK，並選擇一個目錄存放

5. 執行 `flutter doctor` 查看還有什麼需要安裝
   - 從 <https://developer.android.com/studio/index.html> 下載 Android SDK
