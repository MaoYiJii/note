# Bootstrap 4

#### row + col 強制換行

```html
<div class="row">
    <div class="col-6"></div>
    <div class="w-100"></div>
    <div class="col-6"></div>
</div>
```

#### 水平置中

```html
<div class="d-flex justify-content-center">
```

#### 垂直置中

```html
<div class="d-flex align-items-center">
```

#### 水平置中 + 垂直置中

```html
<div class="d-flex align-items-center justify-content-center">
```
